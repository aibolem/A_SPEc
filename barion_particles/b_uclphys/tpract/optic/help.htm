Получение экспериментальных данных и расчет по оптической модели

Заходим на CDFE

Далее следует задать параметры рассеяния. Для данной задачи обязательными являются следующие пункты:

    1. материал мишени (средние и тяжёлые не радиоактивные элементы),
    2. налетающая частица,
    3. тип реакции – elastic scattering,
    4. характер выводимых экспериментальных данных – Angular distribution, general.

Остальные параметры можно не указывать.

Нажать кнопку Search.

На открывшейся странице выбрать в списке реакций подходящую по энергетическому и угловому диапазону. Для выполнения задачи нужно получить графики при нескольких значениях энергии и для широкого диапазона углов (примерно от 0 до 120 – 160). Для просмотра экспериментальных данных в формате EXFOR можно перейти по ссылке “the source data file”.

Следует обратить внимание на наличие погрешностей на графике и в исходном коде EXFOR. Этот файл можно сохранить полностью либо вырезать только ту часть, где хранится таблица углового распределения.

  1. Зайти на сайт NRV.
  2. Выбрать налетающую частицу и частицу мишени.
  3. Указать систему координат и энергию реакции.
  4. По желанию изменить или оставить параметры оптического потенциала. Приближённые значения параметров оптического потенциала можно посчитать по формулам, приведённым на сайте NRV (C.M. Perey, F.G. Perey “COMPILATION OF PHENOMENOLOGICAL OPTICAL-MODEL PARAMETERS 1969 - 1972”). Для перехода к этому сайту нажмите кнопку OMP systematics.
  5. Нажать на кнопку PREPEAR.
  6. В отрывшемся окне ввести экспериментальные данные, полученные из пункта 1. При вводе данных нужно обратить внимание на их единицы измерения: сечения – в mb (милибарнах), углы – в градусах, а погрешности – в процентах.

    Либо здесь можно нажав кнопку «Обзор», ввести экспериментальные данные из сохраненного на предыдущем этапе файла. Затем следует нажать «IMPORT».

  7. Сохранить эти данные («Submit»).
  8. Нажать на кнопку Open на основной странице, выбрать сохраненный файл. Вы увидите экспериментальный график.
  9. Нажать на кнопку Calculate. Рассчитанный результат вы увидите в новом окне.
  10. Дальше можно заняться подгонкой параметров. Их можно подгонять вручную или автоматически. Для автоматической подгонки нужно установить флажки на тех параметрах, которые вы хотите уточнить и нажать кнопку Calculate. Уточнённые значения выбранных параметров показаны красным в открывшемся окне.
